## Overview
This edge AI project demonstrates the power of TensorFlow Lite for Microcontrollers (TFLite Micro). It dynamically captures 96x96 images via an Arducam, runs an INT8-quantized MobileNetV2 inference to detect defects, and physically actuates a servo mechanism to sort objects on a conveyor belt—all without any internet connectivity.
